# PHP Desktop
Create GUI applications for Linux systems using php.

## Installation
- Clone repository
  ```bash
  git clone https://github.com/grephq/php-desktop.git
  ```
- Enable extension loading in ```php.ini```
  ```ini
  enable_dl = On
  ```
- Run ```setup.sh```
